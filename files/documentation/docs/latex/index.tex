\hypertarget{index_autotoc_md2}{}\doxysubsection{Description}\label{index_autotoc_md2}
This project develops an innovative interactive tool designed to offer a real-\/time audiovisual experience for users. The envisioned objective of \mbox{\hyperlink{classIAV}{IAV}} is to provide individuals with limited mobility or minimal musical background an accessible way to engage with positive musical stimuli, using simple actions like hand movements, to inspire and motivate them to participate in creative expression.

\hypertarget{index_autotoc_md15}{}\doxysubsubsection{Starting the Experience}\label{index_autotoc_md15}

\hypertarget{index_autotoc_md16}{}\doxysubsubsection{Exiting the Application}\label{index_autotoc_md16}

\hypertarget{index_autotoc_md17}{}\doxysubsubsection{Application Workflow}\label{index_autotoc_md17}

\begin{DoxyEnumerate}
\item A 5-\/second countdown timer initiates the experience.
\item The webcam captures a frame to initialize the tracking algorithm.
\item The interactive experience begins\+:
\begin{DoxyItemize}
\item The tracking algorithm continuously updates the audiolizer.
\item The audiolizer streams audio and shares audio data with the visualizer.
\item The visualizer updates in real-\/time, responding to both visual and audio inputs.
\end{DoxyItemize}
\item After 10 seconds, the cycle restarts from the beginning.
\item If the tracking algorithm fails, the application automatically restarts the cycle.\textquotesingle{}
\end{DoxyEnumerate}
