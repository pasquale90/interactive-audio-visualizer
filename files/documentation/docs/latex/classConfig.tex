\doxysubsection{Detailed Description}
Singleton class to manage configuration settings, providing a unique point of access to the configuration settings. 

\begin{DoxyNote}{Note}
The \mbox{\hyperlink{classConfig}{Config}} class is responsible for reading settings from the settings database and initializing objects of different configuration types. 

It uses a map to store settings and provides methods to read and write them. 
\end{DoxyNote}

\mbox{\hyperlink{classSettingsDB}{Settings\+DB}} class for managing settings using a SQLite database. 
\end{DoxySeeAlso}
