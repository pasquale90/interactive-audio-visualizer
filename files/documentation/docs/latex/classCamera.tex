\doxysubsection{Detailed Description}
Class representing a camera object. 

\begin{DoxyNote}{Note}
This class encapsulates the functionality for capturing frames from a camera. 

The \mbox{\hyperlink{classCamera}{Camera}} class is responsible for initializing the camera and capturing frames. 

The \mbox{\hyperlink{classCamera}{Camera}} class uses Open\+CV library. 
\end{DoxyNote}
